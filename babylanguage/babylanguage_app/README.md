# Baby Language App

Early language learning app for children 0-5 years. Help your child find their voice through age-appropriate language activities, milestone tracking, and AI-powered insights.

## Features

- 📚 **120+ Language Milestones** across 7 developmental categories
- 🎯 **Daily Activity Suggestions** personalized by age and progress
- 📊 **Progress Tracking** with charts and insights
- 🤖 **AI Weekly Advice** using GPT-4
- 🔥 **Streak Gamification** to build consistent habits
- 📱 **2-Tab Navigation** for simplicity (Dashboard/Advice, Milestones)

## Language Categories

1. **Early Communication & Social** - Eye contact, gestures, turn-taking
2. **Receptive Language** - Understanding words and instructions
3. **Expressive Language** - First words, phrases, sentences
4. **Vocabulary & Concepts** - Categories, opposites, descriptive words
5. **Phonological Awareness** - Rhyming, syllables, sound discrimination
6. **Emergent Literacy** - Book handling, letter knowledge, comprehension
7. **Pragmatics & Conversation** - Questions, topic maintenance, emotions

## Getting Started

### Prerequisites

- Flutter SDK (>=2.17.0)
- Firebase project with Firestore and Authentication enabled
- FlutterFire CLI installed

### Setup

1. **Clone and install dependencies**

```bash
cd babylanguage_app
flutter pub get
```

2. **Configure Firebase**

```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure Firebase for this app
flutterfire configure --project=baby-language --out=lib/firebase_options.dart
```

3. **Add Firebase configuration files**
   - Android: Place `google-services.json` in `android/app/`
   - iOS: Place `GoogleService-Info.plist` in `ios/Runner/`

4. **Deploy Firestore rules and indexes**

```bash
cd ..  # Navigate to babylanguage folder
firebase deploy --only firestore:rules
firebase deploy --only firestore:indexes
```

5. **Import sample milestones**
   - Use Firebase Console or a script to import `sample_language_milestones_seed.json`

6. **Run the app**

```bash
flutter run
```

## Firebase Collections

- `babies` - Child profiles
- `language_milestones` - Public milestone content (read-only)
- `activity_logs` - Completed activities
- `milestone_completions` - Achieved milestones
- `daily_activity_suggestions` - AI-generated daily activities
- `weekly_progress_summaries` - Weekly insights
- `user_streaks` - Gamification streaks
