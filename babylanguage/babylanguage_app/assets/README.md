# Assets Folder

Contains images, icons, and fonts for the Baby Language app.

## Structure

- `images/` - App images, illustrations, onboarding graphics
- `icons/` - Custom icons and category icons
- `fonts/` - Custom fonts (if any)

## To Add

1. **Onboarding images** - Visual guides for language activities
2. **Category icons** - Icons for each of the 7 language categories:
   - Early Communication & Social
   - Receptive Language
   - Expressive Language
   - Vocabulary & Concepts
   - Phonological Awareness
   - Emergent Literacy
   - Pragmatics & Conversation
3. **App icon** - 1024x1024 app icon for iOS/Android
4. **Splash screen** - Launch screen graphics

## Icon Generation

Use a tool like:
- https://appicon.co/
- https://www.appicon.build/

to generate all required sizes from a single 1024x1024 image.
